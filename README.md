# Исследовательский анализ бизнес-показателей маркетплейса

## Описание

Данный репозиторий содержит проект, в котором я применяю навыки использования Python, Pandas, Matplotlib, Seaborn, Scipy для исследования ключевых метрик маркетплейса и метрик unit-экономики. Поиск точек роста и подведение результатов эксперимента, проводимого сторонним аналитиком, также является частью работы.

## 📈 Выводы

В итогам проведенного исследования было обнаружено, что маркетплейс терпит убытки с определенного канала привлечения пользователей. Найдена проблемная зона воронки пользовательского пути на мобильных устройствах, выявлена проблема с удержанием пользовательских сегментов regular и medium.

По результатам проведенного А/В эксперимента по внедрению бонуса за первую покупку пользователям определенного канала можно сказать, что он успешен, но требует поэтапного внедрения с мониторингом долгосрочных эффектов unit-экономики.
