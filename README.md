# Исследовательский анализ бизнес-показателей маркетплейса
Данный репозиторий содержит проект, в котором я применяю навыки использования Python, Pandas, Matplotlib, Seaborn, Scipy для исследования ключевых метрик маркетплейса и метрик unit-экономики. Поиск точек роста и подведение результатов эксперимента, проводимого сторонним аналитиком, также является частью работы.

В результате проведенного исследования было обнаружено, что маркетплейс терпит убытки с определенного канала привлечения пользователей. Найдена проблемная зона воронки пользовательского пути на мобильных устройствах, выявлена проблема с удержанием пользовательских сегментов regular и medium.

По результатам проведенного эксперимента по внедрению бонуса за первую покупку пользователям определенного канала можно сказать, что он успешен, но требует поэтапного внедрения с мониторингом долгосрочных эффектов unit-экономики.

Данные, по которому была проведена исследовательская работа, была предоставлена образовательной платформой Яндекс.Практикум в рамках обучения по программе "Аналитик данных. Расширенный курс".

Для выполнения проекта в области анализа данных с иcпользованием Python необходимы библиотеки Pandas, Matplotlib, Seaborn, Scipy, Numpy.
